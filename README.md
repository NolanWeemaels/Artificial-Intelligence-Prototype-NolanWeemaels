# Klopt Da? AI

Klopt Da? AI is an AI-powered feedback tool for graphic designers and students.  
Users can upload a Web/App or Print design and receive automatic feedback about hierarchy, composition, spacing and visual structure.  
The application uses the OpenAI Vision API together with a small RAG system containing design principles and UI/UX rules.

---

## Up & running

### 1. Open the project folder

Open the project in VS Code or another code editor.

### 2. Install dependencies

```bash
npm install
```

### 3. Create a `.env` file

Create a `.env` file in the root folder and paste this inside:

```env
OPENAI_API_KEY=your_openai_api_key
```

### 4. Start the server

```bash
node server.js
```

### 5. Open the project

Open in browser:

```txt
http://localhost:3000
```

## Features 

- Upload Web/App designs
- Upload Print designs
- AI-generated hierarchy feedback
- AI-generated composition feedback
- Component recognition using AI Vision
- Visual overlay system
- RAG integration using local design knowledge
- Separate analysis flows for Web/App and Print
- Branded custom UI
